填写课程接口信息

先在浏览器里登录 {{ platform_name }}，进入“我的课程”页面，从开发者工具 Network 里复制课程列表接口。 系统只使用你主动提供的 Cookie 调用接口读取课程信息，不保存平台账号密码。

如何复制接口

1在浏览器正常登录 {{ platform_name }}。

{% if platform_name == '中国大学 MOOC' %}

2进入“我的课程 -> SPOC课程”，只找返回 SPOC 课程列表的接口。

{% elif platform_name == '学习通' %}

2进入“课程 -> 我学的课”，只找返回我学课程列表的接口。

{% elif platform_name == '智慧树' %}

2进入“智慧课程 -> 进行中”，只找返回进行中课程列表的接口。

{% elif platform_name == '雨课堂' %}

2进入“课程班级 -> 我听的课”，只找返回我听课程列表的接口。

{% else %}

2进入能看到自己课程的页面，只找返回本人课程列表的接口。

{% endif %}

3按 F12 打开开发者工具，切到 Network / 网络。

4筛选 Fetch/XHR，然后刷新页面。

5找到包含 course、clazz、mycourse、studentcourse、classroom、lesson 等关键词的请求。

6复制 Request URL、Cookie，POST 接口还要复制 Form Data / Payload。

7保存后回到平台绑定页点击“读取课程”。
